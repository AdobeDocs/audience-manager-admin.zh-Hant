# 公司 {#companies}

使用 [!UICONTROL Companies] 頁面，查看Audience Manager配置中的公司清單。 您可以編輯或刪除現有公司或建立新公司，前提是您分配了相應的用戶角色。

![](assets/companies.png)

通過按一下所需列的標題，可以按升序或降序對每列進行排序。

使用 [!UICONTROL Search] 或清單底部的分頁控制項，以查找所需的公司。
