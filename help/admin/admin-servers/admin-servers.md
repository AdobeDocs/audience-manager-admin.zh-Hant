# 伺服器概述 {#servers-overview}

使用 [!UICONTROL Servers] 頁面檢視Audience Manager設定中的伺服器清單。 您可以編輯或刪除現有伺服器，或建立新伺服器，前提是您已指派適當的使用者角色。

<!-- c_servers.xml -->

![](assets/servers.png)

您可以按一下所需欄的標頭，以遞增或遞減順序排序每個欄。 使用 [!UICONTROL Search] 方塊或清單底部的分頁控制項來尋找所需的伺服器。
