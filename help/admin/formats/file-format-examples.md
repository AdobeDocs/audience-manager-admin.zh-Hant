# 檔案格式巨集範例 {#file-format-macro-examples}

如何使用宏建立出站、 [!DNL FTP] 檔案模板。

>[!NOTE]
>
>在桌子上， **粗臉** type標識每個宏及其相關輸出。 對於格式示例，已添加&lt; >符號，以幫助直觀地分隔每個宏。
