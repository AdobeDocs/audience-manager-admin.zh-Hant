# 將入口網站使用者指派給Tableau使用者 {#assign-a-portal-user-to-tableau-user}

<!-- t_tabeau.xml -->

使用[!UICONTROL Reporting]頁面讓入口網站使用者成為[!DNL Tableau]使用者。 這可讓使用者在Audience Manager中檢視[!DNL Tableau]報表。

1. 按一下&#x200B;**[!UICONTROL Reporting]** > **[!UICONTROL Assign Portal User to Tableau User]**。

   ![](assets/tableau.png)

1. 若要指派使用者，請在所需的合作夥伴列中，在文字欄位中輸入[!DNL Tableau]使用者名稱，然後按一下文字欄位外部。

若要刪除使用者指定，請在所需的合作夥伴列中，從文字欄位中移除使用者名稱，然後按一下文字欄位外部。
