# 報表 {#reporting}

透過建立新報告或編輯或刪除現有報告來管理Audience Manager報告。 您也可以將入口網站使用者指派為 [!DNL Tableau] 使用者。

<!-- c_reporting.xml -->

![](assets/reporting.png)

您可以按一下所需欄的標頭，以遞增或遞減順序排序每個欄。

使用 [!UICONTROL Search] 方塊或清單底部的分頁控制項來尋找所需的報表。
